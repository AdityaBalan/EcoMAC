# EcoMAC — Energy-Efficient Neural Accelerator

An FPGA-based neural network inference accelerator combining:
- Approximate Computing (LOA Multiplier)
- Sparse-Aware MAC Scheduling (CSR Format)
- Digital Compute-in-Memory (Row-Broadcast Architecture)

## Target Hardware
PYNQ-Z2 (Zynq-7000 SoC) | 50 MHz | Vivado 2023.x
